# CanopyOps: boundaries

These limits govern use even when installation and invocation succeed. Return consequential authority to the user where stated.

## canopyops

Pesticide/PGR, quarantine/destruction, major setpoint, CO2/life-safety, compliance, inventory, release, money, and irreversible actions require accountable-human approval; extraction, illegal-market, and evasion work is declined.

Static package validation does not establish live tool health, external truth, publication, or customer outcome. See the [host evidence boundary](HOST-EVIDENCE-BOUNDARY.md).
