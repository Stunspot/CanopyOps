# CanopyOps v0.1.3 release notes

Release date: 2026-07-18

CanopyOps v0.1.3 is the dual-host packaging release. It preserves the v0.1.2 cultivation intelligence, safety boundaries, workflows, deterministic utilities, schemas, and evaluation cases while making the same complete skill directly installable in Codex, Claude.ai, and Claude Code.

## What changed

- Added `claude-ai/canopyops-v0.1.3.zip` as a portable one-skill Claude.ai upload.
- Shortened the `SKILL.md` discovery description to the current Claude metadata limit without changing the operating instructions.
- Added non-coder Claude.ai installation, activation, verification, and recovery guidance.
- Advanced the Codex plugin package version to `0.1.3`.

## Verification

- The standalone `canopyops/` tree passes the current Augment Builder Codex and Claude profiles.
- The plugin-bundled skill matches the standalone tree byte-for-byte.
- The Claude ZIP uses portable forward-slash paths, contains one top-level `canopyops/` folder, and matches the standalone tree byte-for-byte after extraction.
- All 12 deterministic tests pass.

## Evidence boundary

Live Claude.ai upload, enablement, activation, progressive resource loading, script execution, and persistence were not exercised for this release. Live Claude Code execution, field use, equipment integration, current jurisdiction coverage, representative-user testing, and formal accessibility or legal assessment also remain unverified. Structural compatibility is not a live-host or field-fitness claim.
