# Jurisdiction Packs

No maintained jurisdiction pack ships in v0.1. CanopyOps supplies a verification method because regulations, license conditions, pesticide permissions, testing limits, local ordinances, and reporting systems change.

A future pack must identify its issuing authorities, jurisdiction and license scope, effective/access dates, maintainer, update cadence, source records, known exclusions, and expiry/revalidation rule. It must never silently convert an example into current legal authority.
