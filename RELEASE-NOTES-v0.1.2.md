# CanopyOps v0.1.2 Release Notes

Release date: 2026-07-17

CanopyOps v0.1.2 is the public-finalization release of the Collaborative Dynamics Augment for lawful cannabis cultivation planning, diagnostics, calculations, harvest review, compliance verification, and auditable operating records.

## What changed since v0.1.1

- Removed stale private-release and release-candidate language from public documentation.
- Added a direct GitHub installation path near the top of the repository README.
- Regularized the Codex plugin version as `0.1.2`.
- Updated privacy, security, support-status, issue-template, and release references for the public distribution.
- Rebuilt standalone, plugin, public-repository, and OpenAI upload archives from one canonical skill tree.

The runtime skill instructions, persona, workflows, safety boundaries, calculations, schemas, examples, and behavioral cases are unchanged from v0.1.1.

## Included

- One complete `canopyops` Agent Skill with the Ella Greenfield v2 cultivation persona.
- Five routed workflows covering crop planning, incidents, environment/root zone, harvest/quality, and compliance/operations.
- Independent review-and-release workflow.
- Twelve reusable operational templates and six machine-readable schemas.
- Nine deterministic Python utilities for calculations, normalization, validation, linting, freshness checks, and packaging.
- Four worked demonstrations and canonical behavioral evaluation cases.
- Codex/local, Claude Code, chat, and bot adapters.
- Branded Codex plugin and Collaborative Dynamics marketplace entry.
- Customer onboarding, installation, safety, examples, FAQ, support, security, contribution, privacy, licensing, and release documentation.
