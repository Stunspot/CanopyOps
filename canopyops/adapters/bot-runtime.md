# Bot Runtime Adapter

Use the same single-skill activation and workflow routing. Store operational artifacts in an approved persistent workspace; conversation history alone is not canonical state. Restrict integrations to read-only retrieval unless a separately authorized system defines writes. Never map CanopyOps recommendations directly to actuators, seed-to-sale records, pesticide systems, or batch release.

Every handoff must carry current state, active hold/risk, authority status, next action/owner, and re-entry condition.
